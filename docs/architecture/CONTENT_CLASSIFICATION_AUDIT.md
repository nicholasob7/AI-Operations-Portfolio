# Content Classification Audit

## Status

This document records the current planning-level classification judgment for the site's public content surfaces.

It is an architectural audit, not yet the implemented canonical content model.

Its purpose is to guide later canonical-truth and projection work without prematurely converting planning judgments into implementation structures.

## Surface Role Classification

### Homepage

Primary role:

- navigation/reference
- summary

The homepage orients the user, establishes public identity and direction, and routes the user toward deeper destination surfaces.

### Resume

Primary role:

- summary

The resume synthesizes broader capability, experience, qualifications, and public profile across the site.

### Complete

Primary role:

- primary detail

This surface carries detailed project-specific material for the remediation project and functions as a major evidence-bearing surface.

### Active

Primary role:

- primary detail

This surface carries detailed project-specific material for the migration/stabilization project and functions as a major evidence-bearing surface.

### Quality

Primary role:

- interpretive/support

This surface supports and contextualizes broader capability claims rather than replacing the primary record.

### Personal

Primary role:

- interpretive/support

This surface supports and contextualizes broader public claims and positioning rather than serving as the main factual record.

## Current Audit Judgment

The site is projection-heavy.

Most current meaningful blocks fall into:

- summary projection
- evidence/support
- support projection

Direct fact-bearing blocks exist, but the current public prose is dominated by projections of broader truth rather than by prose that itself deserves canonical status.

## Governed Prose Judgment

Current governed-prose candidate set is effectively empty or near-empty.

At present, the audit does not identify a large body of public prose whose exact wording clearly crosses the governed-prose threshold.

This does not mean no future governed prose will exist.
It means the current site should be assumed to rely primarily on:

- normalized facts
- relationships
- evidence/support records
- projectional summaries/details/support

## Evidence-Bearing Surfaces

The strongest current evidence-bearing surfaces are:

- complete
- active

These surfaces carry the densest project-specific supporting material and should be treated as the strongest current source of detail-level public substantiation.

## Summary of Planning Position

At planning level, the site currently appears to operate as:

- homepage: orienting and summarizing projection
- resume: synthetic summary projection
- complete/active: primary detail and evidence-bearing projections
- quality/personal: interpretive and support projections

This audit is a planning judgment only.
It does not yet create implementation truth objects, schemas, or content structures.
It guides later canonical extraction, evidence modeling, and projection work, but is not itself the implemented content model.
